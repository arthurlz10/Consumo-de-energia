## 📌 Sobre o projeto
Este é um projeto simples desenvolvido em **Python**, com o objetivo de calcular o **consumo mensal de energia elétrica** de um aparelho.

Ele foi criado para praticar o conceito de programação baseado no ciclo:
➡️ **Entrada → Processamento → Saída**

---

## 🎯 Funcionalidades
- Recebe o **nome do aparelho**
- Recebe a **potência (em watts)**
- Recebe o **tempo de uso diário (em horas)**
- Calcula o **consumo mensal em kWh**
- Exibe o resultado formatado
- Calcula o **custo estimado de energia**

---

## 🧮 Fórmula utilizada
consumoMensal = (potencia × horasDia × 30) / 1000
